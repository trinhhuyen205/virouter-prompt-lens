Privacy Policy

Extension: Virouter Prompt Lens
Last updated: June 2026

What data we collect

Virouter Prompt Lens does not collect, transmit, or store any personal data on remote servers operated by this extension.

  • Images: Images you choose to analyze are resized locally in your browser and sent directly to the OpenAI-compatible API endpoint you configure (default: api.virouter.com). The extension itself never stores or forwards images to any other destination.
  • API key: Your API key is stored locally in chrome.storage.local on your device only and is sent only to your configured API endpoint as an Authorization header.
  • Prompt history: Analysis history is stored locally on your device only. You can clear it at any time from the popup or panel.
  • No analytics, no tracking: The extension contains no analytics, telemetry, crash reporting, or tracking code of any kind.

Optional host access permission

If you enable "Show hover button on all sites", the extension requests access to all HTTP/HTTPS pages to inject a lightweight UI overlay. This permission is optional, granted by you explicitly, and can be removed at any time from extension settings. It is used solely to display the prompt button — no page content is read or transmitted.

Third-party API

When you analyze an image, the resized image data and your prompt are sent to the API base URL you configure. By default this is api.virouter.com. Please refer to Virouter's own privacy policy for how that service handles data. You may point the extension at any OpenAI-compatible API of your choice.
